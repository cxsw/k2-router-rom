# 你也可以自动编译
 1. fork本项目
 2. 在自己的代码库的 **Actions** 中执行 **Padavan固件编译工作流** 
 3. 25分钟后,就可以在自己的 **release** 中得到全新编译的固件(**k2p,k2**).
 4. 脚本流程是: 自动下载**hanwckf**/rt-n56u.git源码然后利用github的虚拟机资源自动编译k2p,k2,并发布到release.
